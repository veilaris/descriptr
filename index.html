Как заполнить ↓

Каждая характеристика — с новой строки в формате Название: значения

  • / — только одно значение из списка:
    Тип: холодильник / стиральная машина / духовой шкаф — выбирается что-то одно
  • , — можно выбрать несколько значений:
    Цвет: чёрный, красный, белый, бежевый — можно указать 2 и более
  • Числовые поля — пишите единицу в значении:
    Объём: число (л)
  • Текстовые поля — допустим любой произвольный текст:
    Комплектация: текст

Пример:

Тип: холодильник / стиральная машина / духовой шкаф
Цвет: чёрный, красный, белый, бежевый
Объём: число (л)
Комплектация: текст
Формат JSON-файла ↓

Каждый ключ — название характеристики, значение — допустимые варианты:

  • / — только одно значение из списка:
    "Тип": "холодильник / стиральная машина / духовой шкаф" — выбирается что-то одно
  • , — можно выбрать несколько:
    "Цвет": "чёрный, красный, белый, бежевый" — можно указать 2 и более
  • Числовые поля:
    "Объём": "число (л)"
  • Текстовые поля — допустим любой произвольный текст:
    "Комплектация": "текст"

Пример файла:

{
  "Тип": "холодильник / стиральная машина / духовой шкаф",
  "Цвет": "чёрный, красный, белый, бежевый",
  "Объём": "число (л)",
  "Комплектация": "текст"
}
